# media-notas-portugol
Média de Notas em Portugol
Descrição

Este projeto foi desenvolvido para praticar lógica de programação utilizando Portugol (Visualg).

O programa solicita ao usuário várias notas, calcula a média e informa se o aluno foi aprovado ou reprovado.

# Conceitos utilizados
- Variáveis
- Entrada e saída de dados
- Operações matemáticas
- Estruturas condicionais (se/senão)

# Como executar
1. Abra o Visualg
2. Copie o código do projeto
3. Execute o algoritmo

Objetivo

Praticar lógica de programação e fortalecer a base para linguagens futuras.
